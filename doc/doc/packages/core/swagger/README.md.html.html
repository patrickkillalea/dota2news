  <h1>
    <a href="#swagger%20for%20mean.io" name="swagger%20for%20mean.io" class="pilcrow">&#182;</a>
    Swagger for Mean.io
  </h1>
</div>


<p>This package is really very basic but it does provide the Swagger endpoints to 
document your API.  It also hosts the Swagger-UI package which is available at 
/docs/ by default.</p>

<p>To get things up and running here is what you need to know:</p>

<ul>
<li>The models are currently only defined in the <code>models.js</code> file. Each package should
have its own /docs folder and in that you can sepcify the models.js</li>
<li>Specify each service in the services.js file within the /docs of the relevent package
See examples below</li>
</ul>


<div class="pilwrap" id="getting%20the%20articles%20example%20working">

* <code>swagger.add(__dirname);</code> to enable swagger support
Ensure <code>swagger.add</code> is AFTER <code>package.routes</code></p>
